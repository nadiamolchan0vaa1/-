# Политика конфиденциальности для приложения BookClab

**Дата вступления в силу:** 19.01.2025  
**Дата последнего обновления:** 19.01.2025  

## 1. Общие положения

Настоящая политика конфиденциальности (далее — "Политика") описывает, как приложение **BookClab** (далее — "Приложение"), разработанное **Надей Семечко**, собирает, использует, хранит и защищает информацию пользователей.

Приложение распространяется через маркетплейс **NashStore** и предназначено для пользователей мобильных устройств на платформе Android.

## 2. Сбор информации

### 2.1. Собираемая информация включает:

**Персональные данные:**
- Адрес электронной почты (при регистрации через Firebase Authentication)
- Имя пользователя (опционально, для отображения в приложении)
- Аватар профиля (опционально, при загрузке из галереи устройства)

**Технические данные:**
- Уникальный идентификатор устройства (для Firebase Analytics)
- Версия операционной системы Android
- Информация об использовании приложения (аналитика)

**Контент пользователя:**
- Списки книг на полках
- Избранные книги
- Отзывы и рейтинги книг
- История поиска

### 2.2. Не собирается:
- Финансовая информация
- Контакты телефонной книги
- Точное местоположение
- Данные других приложений

## 3. Использование информации

Собранная информация используется для:

### 3.1. Основных функций:
- Регистрация и аутентификация пользователей
- Хранение пользовательских данных (книжные полки, избранное)
- Персонализация интерфейса приложения
- Восстановление данных при переустановке приложения

### 3.2. Улучшения работы приложения:
- Анализ использования функций приложения
- Выявление и исправление ошибок
- Разработка новых функций на основе статистики использования

### 3.3. Коммуникации:
- Уведомления о новых функциях (только внутри приложения)
- Ответы на обращения пользователей через обратную связь

## 4. Хранение и защита данных

### 4.1. Период хранения:
- Данные хранятся до момента удаления аккаунта пользователем
- Технические логи хранятся не более 90 дней
- Резервные копии данных хранятся 30 дней после удаления

### 4.2. Места хранения:
- **Firebase Authentication** - аутентификационные данные
- **Firebase Firestore** - пользовательский контент (Россия, сервера Google)
- **Android DataStore** - локальные настройки на устройстве

### 4.3. Меры защиты:
- Шифрование передаваемых данных (HTTPS/TLS)
- Авторизованный доступ к базам данных
- Регулярное обновление библиотек безопасности
- Резервное копирование данных

## 5. Передача данных третьим лицам

### 5.1. Сервисы, используемые в приложении:

**Google (США):**
- Firebase Authentication - управление пользователями
- Firebase Firestore - хранение данных
- Google Books API - поиск книг
- Firebase Analytics - аналитика использования

**Условия передачи:**
- Передача осуществляется в обезличенном виде
- Данные защищены соглашениями о конфиденциальности
- Соответствие GDPR и требованиям российского законодательства

### 5.2. Не передаются:
- Списки контактов пользователей
- Переписки и личные сообщения
- Коммерческая информация

## 6. Права пользователей

Каждый пользователь имеет право:

### 6.1. Доступ к данным:
- Просматривать свои данные через интерфейс приложения
- Экспортировать свои коллекции книг в формате JSON
- Получать информацию о хранимых данных по запросу

### 6.2. Управление данными:
- Редактировать профиль в любое время
- Удалять отдельные книги из коллекций
- Отключать сбор аналитических данных
- Экспортировать все свои данные

### 6.3. Удаление данных:
- Удалить аккаунт через настройки приложения
- Запросить полное удаление всех данных через обратную связь
- После удаления данные не подлежат восстановлению

## 7. Файлы cookie и аналогичные технологии

Приложение использует:

### 7.1. Локальное хранилище:
- **DataStore** - для хранения настроек и предпочтений
- **SharedPreferences** - для кэширования данных
- **Firebase SDK** - для идентификации устройства

### 7.2. Отслеживание:
- Аналитика использования функций приложения
- Статистика ошибок и сбоев
- Оптимизация производительности

## 8. Изменение политики конфиденциальности

### 8.1. Уведомление об изменениях:

### 8.2. Право отказа:

### 9.2. Родительский контроль:
